# EasySOA Demo 0.3

## Prerequisites

* Sun's 1.6 Java JDK
* The `JAVA_HOME` variable set to your JDK folder

NB: If you are using a virtual machine, make sure its RAM is 1GB+.

## Setup

1. Set your browser's HTTP proxy to "127.0.0.1:8081"
  - Make sure to remove any proxy exceptions
  - Only set the proxy to the HTTP protocol
2. Use `run.sh` (or `.bat`) or launch the servers manually
3. If the script doesn't show a browser automatically, browse to "127.0.0.1:8083/index.html"
